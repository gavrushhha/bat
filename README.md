# Инструкция

* Склонировать репозиторий
* Добавить в Config.ini два значения
1. SourceFolder - Локальная папка, которую нужно мониторить на наличие новых файлов
SourceFolder=C:\Users\sofja\monitor
2. DestinationFolder - Сетевой путь (UNC) или локальный путь, куда копируются файлы
DestinationFolder=C:\Users\sofja\Desktop\winlav

* Запустить в терминале файл RunRobocopyMonitor.bat
